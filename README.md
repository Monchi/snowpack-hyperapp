# snowpack-hyperapp

Simple ToDo list built with [snowpack](https://www.snowpack.dev/) and [hyperapp](https://hyperapp.dev/).

You can see live demo [here](https://snowpack-hyperapp.netlify.com).

## Build

```sh
npm build
```

Then access to dist/index.html.
